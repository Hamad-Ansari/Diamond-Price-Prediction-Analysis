## üéØ Project Overview

This project demonstrates end-to-end data science workflow on the famous Diamonds Dataset (53,940 records) with:

- **Regression**: Price prediction based on diamond features
- **Classification**: Cut quality classification  
- **Comprehensive Analysis**: Feature importance and business insights

## üìä Dataset Features

- **carat**: Diamond weight (0.2 - 5.01)
- **cut**: Quality of cut (Fair, Good, Very Good, Premium, Ideal)
- **color**: Color grade (J (worst) to D (best))
- **clarity**: Clarity measurement (I1, SI2, SI1, VS2, VS1, VVS2, VVS1, IF)
- **depth**: Total depth percentage
- **table**: Width of top relative to widest point
- **price**: Price in US dollars (target for regression)
- **x, y, z**: Dimensions in mm
